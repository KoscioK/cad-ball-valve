# Ball Valve - CAD Design Project

This repository contains a complete CAD design of a ball valve, including 3D models and technical documentation. The project was created as part of a SolidWorks training course.

## Project Overview
The goal of this project was to design a functional ball valve based on specific technical parameters. The documentation includes a main assembly and detailed technical drawings for each individual component.

### Project Tasks:
1.  Preparation of 3D models for all parts based on provided dimensions.
2.  Creation of an assembly drawing with a Bill of Materials (BOM).
3.  Preparation of detailed technical drawings for all 8 components.

## Components List
 The assembly consists of the following parts:
1.  **Main Body** (`glowny_body`) - Material: Brass.
2.  **Ball** (`kula`) - Material: AISI Type 316L Stainless Steel.
3.  **Stem** (`trzpien`) - Material: Brass.
4.  **Lever** (`dzwignia`) - Material: Galvanized Steel.
5.  **Ball Seal** (`uszczelka_kula`) - Material: PTFE.
6.  **Stem Seal** (`uszczelka_trzpien`) - Material: Rubber.
7.  **Screw-in Adapter** (`wkretka`) - Material: Brass.
8.  **Nut** (`nakretka`) - Material: Galvanized Steel.

## Technical Specifications
 The design follows specific dimensions:
*  **Thread (A):** G 1"
*  **Internal Diameter (A1):** 26.0 mm
*  **Total Length (B):** 82.0 mm
*  **Connection Length (C):** 20.5 mm
*  **Height (D):** 61.0 mm
*  **Lever Length (E):** 114.0 mm
